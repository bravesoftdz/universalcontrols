# UniversalControls

Набор компонент, и отработанных решений для разработки на Delphi 2010 и выше.

# Структура проекта

*	**Demo** - проекты демонстрирующие работу этих компонент
*	**Dependent modules** - (зависимые модули). Модули данных зависящие от "чужих" компонент и разработок. Для их работы может потребоваться установка дополнительных модулей в среду разработки.
*	**foreign** - (сторонние) модули сторонних разработчиков включенные как есть.
*	**Useful modules** - внутренние модули компонент. Они самодостаточны и могут работать на чистой IDE
*	**vcl** - модули визуальных компонент.
*	**Versions.ucver** - Версии визуальных компонент и данного пакета компонент в целом
*	**delphi_ver.inc** - версии Delphi для использования в {$IFDEF ....} условиях компиляции
